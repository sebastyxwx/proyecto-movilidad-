# 1. ¿Qué problema soluciona?
* Soluciona el problema de desinformacion
* No saber qué ruta tomar para llegar a un destino
* No tener claridad sobre horarios reales
* No saber cuánto falta para que pase el bus
* No estar informado sobre retrasos
* Pérdida de tiempo esperando transporte

# 2. ¿A qué nivel de decisión impacta?
* Nivel Operativo (principal)
  Afecta decisiones del dia a dia del usuario:
  - Que bus debe tomar
  - A que hora debe salir
  - Donde puede esperar
  - Que retrasos hay
* Nivel Táctico (secundario)
  Puede ayudar a:
  - Empresas de transporte
  - Optimizar tiempos

# 3. ¿Qué decisiones permitirá tomar?
* Elegir la mejor ruta disponible
* Decidir a que hora salir de casa
* Cambiar de ruta si hay retrasos
* Elegir entre distintas empresas de transporte
* Decidir que lugar es más conveniente según ubicación
* Planificar mejor el tiempo de llegada

# 4. ¿Qué pasaría si no existiera ese sistema?
* Los usuarios dependerian solo de horarios estaticos
* Aumentaría el tiempo de espera
* Mas personas llegarían tarde a su destino
* Las empresas no tendrían datos en tiempo real para mejorar el servicio

# Objetivo Gneral del sistema
* Desarrollar una app que brinde información en tiempo real sobre rutas de transporte público, horarios, ubicación del usuario, tiempos estimados de llegada y posibles retrasos, con el fin de optimizar la movilidad y mejorar la experiencia del usuario

# Objetivos Específicos
1. Proporcionar al usuario información precisa sobre qué ruta de bus debe tomar según su ubicación actual y destino.
2. Mostrar en tiempo real el tiempo estimado de llegada del bus, incluyendo notificaciones sobre retrasos o cambios en el servicio.

# Nivel de decisión principal que impacta
* Nivel Operativo: El sistema impacta principalmente en el nivel operativo, ya que ayuda al usuario a tomar decisiones inmediatas y cotidianas como:
   - Qué bus tomar.
   - A qué hora salir.
   - En qué lugar esperar
   - Si cambiar de ruta por retrasos

# Características críticas que debe cumplir
* Para que el sistema sea eficiente, debe cumplir con:
   - Geolocalización precisa del usuario.
   - Información en tiempo real.
   - Actualización constante de datos.
   - Cálculo confiable del tiempo estimado de llegada.
   - Alertas de retrasos o incidencias.
   - Interfaz intuitiva y fácil de usar.
   - Seguridad y protección de datos del usuario

# ¿Qué decisión permitirá tomar?
* Permitirá tomar decisiones como:
  - Elegir la ruta más rápida o conveniente.
  - Decidir la mejor hora para salir.
  - Cambiar de bus si existe retraso.
  - Seleccionar la empresa de transporte más eficiente.
  - Optimizar el tiempo de traslado diario
